# 生信程序员最小知识地图

这份地图不是生物学教科书，而是帮助程序员判断「我该先懂哪些概念，才能正确搭建和维护生信系统」。

## 1. 数据与文件格式层

| 模块 | 必懂对象 | 常见工具 |
|:---|:---|:---|
| 原始测序 | FASTQ、质量值、read layout、barcode、UMI | FastQC、fastp、MultiQC |
| 比对与索引 | SAM/BAM/CRAM、BAI/CSI、reference genome、annotation | BWA、Bowtie2、STAR、minimap2、SAMtools、HTSlib |
| 变异 | VCF/BCF、GVCF、BED、interval、genome build | BCFtools、GATK、DeepVariant、VEP |
| 表达矩阵 | count matrix、TPM/FPKM、feature id、sample metadata | DESeq2、edgeR、limma、Salmon |
| 单细胞 | h5ad、loom、mtx、AnnData、Seurat object | Scanpy、AnnData、Seurat、scvi-tools |
| 蛋白/结构 | FASTA、PDB/mmCIF、MS raw/mzML | UniProt、RCSB PDB、OpenMS、AlphaFold |

## 2. 可复现工程层

| 任务 | 推荐优先级 | 说明 |
|:---|:---:|:---|
| 工作流编排 | 高 | Nextflow/nf-core 或 Snakemake 是多数项目的首选入口 |
| 软件分发 | 高 | Bioconda 和 BioContainers 能显著降低安装与环境漂移成本 |
| 结果汇总 | 高 | MultiQC 是多样本 QC 报告事实标准之一 |
| 云端执行 | 中 | Terra、AnVIL、DNAnexus、Latch 等适合规模化和协作场景 |
| 工作流注册 | 中 | Dockstore、WorkflowHub 适合发布、发现和复用工具/流程 |

## 3. 组学任务层

| 方向 | 核心问题 | 典型组合 |
|:---|:---|:---|
| DNA-seq / 变异分析 | 从 reads 到变异、注释和过滤 | fastp + BWA/minimap2 + GATK/DeepVariant + VEP |
| RNA-seq | 定量、差异表达、富集分析 | STAR/Salmon + DESeq2/edgeR + clusterProfiler/Reactome |
| 单细胞 | QC、降维、聚类、注释、整合 | Scanpy/Seurat + scvi-tools + CELLxGENE |
| 宏基因组 | 物种组成、功能通路、基因组装/注释 | QIIME 2 + Kraken2/MetaPhlAn + HUMAnN + Bakta |
| 蛋白质组 | 质谱处理、鉴定、定量、通路 | OpenMS + FragPipe/MaxQuant + UniProt/Reactome |
| 群体遗传 | GWAS、关联、PRS、稀有变异 | PLINK + Hail + SAIGE/REGENIE + gnomAD |

## 4. 数据库与知识层

| 知识对象 | 首选入口 |
|:---|:---|
| 基因组与注释 | NCBI、Ensembl、UCSC Genome Browser |
| 蛋白与结构 | UniProt、RCSB PDB、AlphaFold DB |
| 变异频率与临床 | gnomAD、ClinVar、dbSNP、cBioPortal |
| 靶点与通路 | Open Targets、Reactome、STRING、KEGG |
| 单细胞公开数据 | CELLxGENE Census、Human Cell Atlas |
| 工具发现 | bio.tools、Bioconda、nf-core、Galaxy ToolShed |

## 5. AI+Bio 层

| 类型 | 代表项目 | 判断重点 |
|:---|:---|:---|
| 结构预测 | AlphaFold、OpenFold | 是否需要 GPU、MSA、模板、数据库下载 |
| 蛋白语言模型 | ESM | embedding、zero-shot、fine-tune 与许可 |
| 结构/序列设计 | RFdiffusion、ProteinMPNN | 任务边界、结构约束、实验验证风险 |
| 变异调用 | DeepVariant | 参考基因组、测序平台、模型版本 |
| 大规模检索 | MMseqs2 | 数据库构建、内存、敏感度与速度取舍 |

## 快速路线

1. 新手工程栈：Biopython + SAMtools/BCFtools + MultiQC + Snakemake。
2. 生产工作流栈：Nextflow + nf-core + Bioconda + BioContainers + MultiQC。
3. 单细胞栈：AnnData + Scanpy + scvi-tools + CELLxGENE，R 侧配 Seurat。
4. 变异分析栈：fastp + BWA/minimap2 + GATK/DeepVariant + Ensembl VEP + gnomAD。
5. AI+Bio 栈：MMseqs2 + AlphaFold/OpenFold + ESM + RFdiffusion/ProteinMPNN。
